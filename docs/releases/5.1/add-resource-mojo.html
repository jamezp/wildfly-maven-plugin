wildfly:add-resource

Full name:

org.wildfly.plugins:wildfly-maven-plugin:5.1.3.Final:add-resource

Description:

Adds a resource

If force is set to false and the resource has already been added to the server, an error will occur and the operation will fail.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
